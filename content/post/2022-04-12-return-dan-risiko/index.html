Pendahuluan

Modul ini membahas secara rinci dua faktor utama dalam mengambil keputusan investasi yaitu return dan risiko. Berbagai jenis return dan risk serta cara penghitungannya juga dibahas di modul ini.

Setelah mempelajari dengan baik modul ini Anda diharapkan mampu menghitung return dan risiko untuk aset tunggal dan portofolio dan konsep diversifikasi. Secara khusus Anda diharapkan mampu:

  • menjelaskan pengertiareturn suatu investasi,
  • menjelaskan pengertian return aset tunggal,
  • menjelaskan return realisasian,
  • menjelaskan return ekspektasian,
  • menjelaskan risiko aset tunggal,
  • menjelaskan koefisien variasi,
  • menjelaskan pengertian return portofolio,
  • menjelaskan return portofolio ekspektasian,
  • menjelaskan pengertian risiko portofolio,
  • menjelaskan risiko portofolio dua aset,
  • menjelaskan risiko portofolio banyak aset, dan
  • menjelaskan pengertian diversifikasi risiko portofolio.
